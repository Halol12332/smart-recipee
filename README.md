Hi. This is how you run the file. 

Open a new terminal, currently you are in SMART_RECIPEE_V2 directory. 
Change directory to backend and run app.py
> cd backend
> python app.py

Open another terminal, change directory to frontend
> cd frontend 
> python -m http.server 8000

Open browser and search
> http://localhost:8000/browse.html

Now you can use the app already.
